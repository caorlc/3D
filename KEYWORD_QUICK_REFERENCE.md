```

❌ 关键词密度超过5%的段落

✓ 台湾用语自然准确
✓ LSI语义词丰富多样

## 常见问题速答

**Q: 如何在不显得生硬的情况下达到关键词密度？**
```
A: 使用语义变体轮流出现
   第1段：產生3D建模成果
   第2段：建立3D模型
   第3段：生成3D設計

   交替使用同义词，避免重复
```

**Q: 如果关键词密度不足怎么办？**
```
A: 不要强行堆砌！改用LSI语义词
   • 多用长尾词和技术词
   • 增加应用场景描述
   • 补充相关问答

   质量 > 数量
```

**Q: 怎样检查是否过度优化？**
```
A: 用「读者视角」阅读：
   1. 能否自然理解？
   2. 是否多次强调同一点？
   3. 句子是否扭曲变形？
   4. 信息是否冗余重复？

   如有任何违和，则已过度优化
```

**Q: 台湾用语不确定时怎么办？**
```
A: 参考本文件的替换表
   不确定的词：
   • 查Google台湾搜索结果
   • 查维基百科繁体中文版
   • 咨询台湾团队成员

   不要机械直译简体
